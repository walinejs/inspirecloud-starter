# [Waline](https://github.com/walinejs/inspirecloud-starter)


Waline 是一款基于 Valine 衍生的简洁、安全的评论系统，该项目是使用[轻服务](https://qingfuwu.cn)部署的一个 [Waline](https://waline.js.org) 应用示例，可以将项目一键部署到轻服务。
